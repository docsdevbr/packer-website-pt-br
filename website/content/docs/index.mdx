# Documentação do Packer

O Packer é uma ferramenta da comunidade que permite criar imagens de máquina
idênticas para diversas plataformas a partir de um único template de origem.
O Packer é comumente usado para criar imagens douradas que times de uma
organização podem usar na infraestrutura de nuvem.

Para instalar o Packer e aprender o fluxo de trabalho padrão do Packer,
experimente os [Tutoriais de Introdução](/packer/tutorials).

## HCP Packer

O registro HCP Packer armazena metadados sobre seus artefatos.
Você pode usar o registro para rastrear informações sobre artefatos de suas
compilações do Packer, designar claramente quais artefatos são apropriados para
ambientes de teste e produção e consultar os artefatos corretos para uso nas
configurações do Packer e do Terraform.

Consulte os seguintes recursos para começar:

- [Documentação do HCP Packer](/hcp/docs/packer)
- [Tutoriais do HCP Packer](/packer/tutorials/hcp-get-started)
- [Acesse o portal do HCP](https://portal.cloud.hashicorp.com)
